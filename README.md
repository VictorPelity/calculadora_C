## CALCULADORA CIENTÍFICA EM C

### Através desse código criei uma calculadora científica sem utilizar a biblioteca <math.h>, desenvolvendo as funções dela com minha própria lógica. Este projeto me permitiu aprofundar meus conhecimentos na linguagem C,  utilizando conceitos como: 
- Loops;
- Funções;
- Recursividade.
