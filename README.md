# 📘 Calculadora Científica em C (Terminal)

Calculadora científica executada no terminal, desenvolvida em linguagem C, sem a utilização da biblioteca <math.h> para fins de treinamento de lógica.

O objetivo do projeto é praticar lógica de programação, funções, recursividade e interação com o usuário via CLI (Command Line Interface).

---

# 🚀 Funcionalidades

## Operações básicas
- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

## Operações matemáticas
- 🔢 Potência
- √ Raiz quadrada (exata)
- ∛ Raiz cúbica (exata)
- ❗ Fatorial (usando recursividade)

## Extras
- Interface simples via terminal
- Limpeza de tela utilizando código ANSI
- Validação básica de erros (ex: divisão por zero, fatorial negativo)

---

# 🧠 Conceitos praticados

Este projeto utiliza conceitos fundamentais da linguagem C:

- Funções
- Recursividade
- Estruturas de decisão (`switch`, `if`)
- Laços de repetição (`for`, `do while`)
- Entrada e saída de dados (`scanf`, `printf`)
- Manipulação de terminal com ANSI escape codes
- Organização de código

---

# 📂 Estrutura do código

| Função | Responsabilidade |
|-------|------------------|
| `soma` | Soma dois números |
| `subtracao` | Subtrai dois números |
| `multiplicacao` | Multiplica dois números |
| `divisao` | Divide dois números |
| `potencia` | Calcula potência usando repetição |
| `fatorial` | Calcula fatorial usando recursividade |
| `raizQuadrada` | Retorna raiz quadrada exata |
| `raizCubica` | Retorna raiz cúbica exata |
| `clear_screen` | Limpa a tela do terminal |
| `calculos` | Menu principal da aplicação |
